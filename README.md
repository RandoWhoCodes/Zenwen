# Zenwen

A tool to help you learn Chinese.

---

### Features (so far)

This is the first version of a very early version of Zenwen.

It doesn't have much yet, but it works.

 - Create & delete vocab decks
 - Create & delete vocab
 - View all your vocab in 1 place
 - Search for decks or vocab in the search bar

Since this is simply just a demo, nothing is saved and all data is temporary. Once you refresh the page everything will be reset.

---

### How it works

Zenwen was built using Flutter and is fully written in dart. Everything runs client side.
